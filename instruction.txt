Особенности сборки проекта:
1. Проект собран с помощью Maven 3.8.8
2. Версия Java 8
3. В проекте использована библиотека Apache Commons IO 2.13.0. Данная библиотека добавлена в зависимости в файле pom.xml

Кроме исходного кода программы в репозиторий выложена исполняемая версия программы MergeSortApp.exe
Конвертация jar-файла в exe-файл была сделана с помощью онлайн-сервиса https://genuinecoder.com/online-converter/jar-to-exe/

Особенности работы программы:
1. Во входных файлах со строковыми данными считаются не валидными:
- пустые строки;
- строки с пробелами;
- строки, значение которых меньше, чем значение предыдущей строки.
Начиная с такой строки все следующие строки не обрабатываются и не попадают в выходной файл.
2. Во входных данных с данными из целых чисел считаются не валидными:
- пустые строки;
- строки с пробелами;
- строки, которые содержат символы, отличающиеся от цифр;
- строки, числа в которых меньше, чем число в предыдущей строки.
- строки, которые содержат целые числа, выходящие за диапазон типа данных Integer.
Начиная с такой строки все следующие строки не обрабатываются и не попадают в выходной файл.
3. Выходной файл получается путем переименования последнего файла, полученного в цикле сортировки файлов, в название файла,
введенного в качестве параметра при запуске программы. Если такое переименование не получается сделать, то программа
оставляет на диске последний файл, полученный в цикле сортировки файлов, и выводит название этого файла в консоль.
